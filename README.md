# dvd-dataVis
A dashboard to visualize and analyze sample dvd rental data.

# Getting Started 
- Fork and Clone this repository
- cd into dvd-dataVis

# Setup / Installation
- To get the backend running on localhost: 4000 :
```bash
node index.js
```

- To get the front-end running on localhost: 3000 :
```bash 
npm start
```
